┌──────────────────────────────────────────────────────────────┐

   ┌────────────────────────────────────────────┐
   │  [🟢 + فاتورة جديدة]  [🔍 بحث...]       │

        ↑

└──────────────────────────────────────────────────────────────┘

   │  └─────────────────────────────────┘ │

   │  │ كشف عيادة       │   1    │  500   │  500.00 ج    ││
   │  └──────────────────┴────────┴────────┴───────────────┘│
   │                                                         │
   │  [+ إضافة عنصر آخر]  [📋 نسخ السطر]                   │
   └─────────────────────────────────────────────────────────┘
   
   مثال لعناصر متعددة:
   ┌─────────────────────────────────────────────────────────┐
   │  1. كشف عيادة         - كمية: 1  - 500 ج               │
   │  2. تحليل دم          - كمية: 1  - 200 ج               │
   │  3. أشعة تلفزيونية    - كمية: 1  - 300 ج               │
   │  ───────────────────────────────────────────────────── │
   │  المجموع الفرعي:                        1000.00 ج     │
   └─────────────────────────────────────────────────────────┘

└──────────────────────────────────────────────────────────────┘

   │  ○ كاش          ○ فيزا                                 │

   └─────────────────────────────────────────────────────────┘


┌──────────────────────────────────────────────────────────────┐

└──────────────────────────────────────────────────────────────┘

   1️⃣ اضغطي زر الحفظ:
      ┌──────────────────────────────────────┐
      │  [إلغاء]  [🟢 حفظ الفاتورة]        │
      └──────────────────────────────────────┘
   
   2️⃣ اضغطي F2 على لوحة المفاتيح
   
   3️⃣ اضغطي Ctrl+S
   
   ✅ تظهر رسالة: "تم إنشاء الفاتورة بنجاح"


╔══════════════════════════════════════════════════════════════╗
║               ⌨️ اختصارات مهمة للسرعة                      ║
╚══════════════════════════════════════════════════════════════╝

   F3        → فتح فاتورة جديدة
   F2        → حفظ الفاتورة
   Ctrl+S    → حفظ (بديل)
   ESC       → إلغاء وإغلاق
   Ctrl+P    → طباعة الفاتورة
